## How to run 
``` bash
## navigate to the file directory
cd STUDENT_COURSE_REGISTRATION_SYSTEM

## run the program using 
python main.py

```
## Classes in this code

- class Person which is a parent to class Student
- Class Student  is the child to class Person
- Class Course which is responsible add course and view course

## Validation Rules Applied

- Student ID and Course ID cannot be empty
- Email must contain `@`
- Phone number cannot be empty
- Course capacity must be a positive integer
- A student cannot register for the same course twice
- Registration is rejected if the course is already full
